# MMPI 해석 보조 프로그램

이미 산출된 MMPI T점수를 입력하면 타당도, 임상척도, 쌍척도/코드타입, 내용척도, 재구성/RC 척도, 보충척도, PSY-5, 긍정적 자원 해석을 한 화면에서 정리하는 로컬 웹앱입니다.
제공된 강의자료의 벡터 어울림 관점을 반영해, 임상척도를 위험 감지, 의미 평가, 정서 처리, 행동 실행, 사회 관계, 정체성 시스템으로 묶어 보는 구조 해석 섹션을 추가했습니다.
3개 이상 상승한 임상척도는 핵심 축, 억제/통제 시스템, 방출/보상 시스템, 내부 순환 구조, 임상 핵심 문제, 개입 전략으로 통합 사례개념화합니다.
TCI는 상담·교육용 설명 프레임으로 NS·HA·RD·P의 높고 낮음 조합에 따른 16가지 기질 유형 별칭을 제공합니다.
TCI 성격 차원은 SD(자율성), CO/Cooperativeness(연대감), ST(자기초월)의 높고 낮음과 조합 패턴을 심층 해석합니다.

- MMPI-2 성인용: `index.html`
- MMPI-A 청소년용: `mmpia.html`
- TCI 성격기질검사: `tci.html`
- 통합 시작 화면: `home.html`

## 사용 방법

1. `home.html`에서 검사 종류를 선택하거나, 성인 사례는 `index.html`, 청소년 사례는 `mmpia.html`을 직접 엽니다.
2. 내담자 정보, 성별/성정체성, 평가 맥락을 입력합니다.
3. 각 탭에서 보유한 T점수를 입력합니다. CNS는 원점수를 입력합니다.
4. 오른쪽 리포트가 자동 생성됩니다.
5. `예시` 버튼으로 샘플 프로파일을 확인할 수 있고, `인쇄` 버튼으로 보고서 초안을 출력할 수 있습니다.

## 중요한 제한

- 이 프로그램은 MMPI 원문 문항, 채점키, 원점수 산출 기능을 포함하지 않습니다.
- 점수는 공식 채점 절차 또는 적법한 검사 시스템에서 산출된 값을 사용해야 합니다.
- 성별 입력은 리포트 맥락 표시용입니다. 원점수를 성별 규준으로 변환하지 않으므로, 입력 T점수는 공식 성별·연령 규준이 이미 반영된 값이어야 합니다.
- 출력 내용은 임상 가설 정리용이며 진단, 감정, 치료 결정의 단독 근거가 아닙니다.
- 타당도 척도에서 해석 제한이 나타나면 임상척도 해석은 보류하거나 매우 조심스럽게 사용해야 합니다.
- 최종 판단은 자격 있는 전문가가 면담, 행동관찰, 병력, 문화적 배경, 의뢰 맥락과 통합해 내려야 합니다.
- MMPI-A 해석은 보호자·교사 보고, 학교 기능, 가족환경, 발달 단계와 반드시 함께 검토해야 합니다.
